## Overview

Library that provides an AES-256 encrypted version of the Android shared preferences

## Setup

#### Gradle

`compile 'com.github.hussainderry'`

#### Maven
```
<dependency>
    <groupId>com.github.hussainderry</groupId>
    <artifactId>-</artifactId>
    <version>0.0.1</version>
</dependency>
```
